## Installation
* Download package 
* Open folder in VS Code 
* ctrl~ (open your terminal) 
* npm install (in terminal)
* install seed.sql

## Usage
After npm install: 
* node index.js (in terminal)
* follow prompts
* database will begin to populate with your additions
